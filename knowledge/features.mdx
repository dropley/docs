# Features

Dropley lets you upload HTML files or static sites and get an instant, shareable URL. No account, no config, no servers.

## Core Capabilities

### Instant Publishing

Upload an HTML file or ZIP archive and receive a public URL in seconds. No build steps, deployment pipelines, or configuration files required.

### Full Static Sites

Publish complete static websites including HTML, CSS, JavaScript, images, and fonts. Add an `index.html` to any folder to create a site — relative paths are preserved and served as-is.

### Interactive Previews

Published artifacts remain fully interactive. Recipients can browse, click, and navigate your site in their browser — just like a live site.

### AI Artifact Support

Paste or upload HTML generated by any AI coding assistant. Dropley is built for the output of LLMs, making it simple to go from prompt to public URL.

### Browser-Based Workflow

Upload directly from your browser. No CLI, no SDK, no software installation, and no account setup required.

### Secure Uploads

Every artifact is isolated with per-artifact boundaries. MIME-type validation and upload protections are applied automatically.

## Publishing Workflow

1. **Generate** — Write your HTML file or generate it with an AI coding assistant.
2. **Upload** — Drag and drop or choose your file or ZIP archive. Files are validated and processed instantly.
3. **Share** — Dropley generates a unique, publicly accessible URL for your artifact.
4. **Send** — Share the link with anyone. No account needed to view.

## Who It's For

- **Developers** — Ship HTML sites, prototypes, and documentation in seconds. No CI setup or deployment config.
- **AI Coding Assistants** — Generate HTML with your AI assistant and publish it instantly.
- **Designers** — Share interactive design prototypes, style guides, and static mockups with a single link.
