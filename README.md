# Dropley Docs

The official documentation website for [Dropley](https://dropley.app) — the easiest way to publish anything generated by software.

This repository contains the content and configuration for [docs.dropley.app](https://docs.dropley.app).

## Tech Stack

- **Site generator**: [Holocron](https://holocron.so) (Mintlify-compatible, Vite plugin)
- **Content format**: MDX
- **Deployment**: Cloudflare Workers via `wrangler deploy`

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to preview the docs site.

## Commands

| Command | Description |
|---------|-------------|
| `npm run dev` | Start local dev server |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |

## Content Guidelines

- Use relative links between pages (e.g., `/knowledge/concepts`)
- Follow the existing MDX component patterns (Callout, Steps, etc.)
- Do not include API endpoint documentation — link to `https://dropley.app/docs/api`
- All content is user-facing, not architectural

## Deployment

Push to `main` triggers automatic build and deployment via GitHub Actions. No manual steps needed.
