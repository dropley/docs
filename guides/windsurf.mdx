# Publishing with Windsurf

Windsurf can publish artifacts to Dropley directly from the IDE.

## Setup

Tell Windsurf to install the Dropley skill:

```
npx skill add dropley/docs
```

## Workflow

1. Use Windsurf's Cascade to generate your project
2. Tell Windsurf: "Publish this to Dropley"
3. Windsurf uses the skill to upload your files and return a shareable URL

## Example

```
Build a project dashboard with index.html, style.css, and app.js.
Use CDN for Chart.js and relative paths. Then publish to Dropley.
```
