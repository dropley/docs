# Publishing with Windsurf

Windsurf (by Codeium) is an AI-powered IDE that can help you create and publish Dropley artifacts.

## Prerequisites

- Windsurf IDE installed

## Workflow

1. Use Windsurf's AI flow to generate your project
2. Create a ZIP from the project directory:
   ```bash
   zip -r output.zip . -x "node_modules/*" ".git/*"
   ```
3. Upload the ZIP to [dropley.app](https://dropley.app)

## Pro Tips

- Windsurf's Cascade can scaffold entire projects from natural language
- Ask Windsurf to build projects with static-file-friendly paths
- Use the built-in terminal for ZIP creation

## Example

```
In Windsurf Cascade: "Build a project dashboard with index.html,
style.css, and app.js. Use CDN for Chart.js and make sure all
paths are relative."
```
