# Publishing with Cursor

Cursor can publish artifacts to Dropley directly from the editor.

## Setup

Tell Cursor to install the Dropley skill:

```
npx skill add dropley/docs
```

## Workflow

1. Use Cursor Composer (Cmd+K) to generate your project
2. Tell Cursor: "Publish this to Dropley"
3. Cursor uses the skill to upload your files and return a shareable URL

## Example

```
Create a personal website with index.html, about.html, styles.css, and images.
Use relative paths. Then publish to Dropley.
```
