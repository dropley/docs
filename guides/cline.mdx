# Publishing with Cline

Cline is a CLI-native AI assistant that can help you build and publish Dropley artifacts from the terminal.

## Prerequisites

- Cline installed and configured

## Workflow

1. Ask Cline to create your project files
2. Have Cline package them into a ZIP:
   ```bash
   zip -r project.zip index.html styles/ scripts/
   ```
3. Upload to [dropley.app](https://dropley.app)

## Pro Tips

- Cline excels at creating files directly in your filesystem
- Ask Cline to create the ZIP as part of its workflow
- Cline can also read the Dropley docs to give you publishing advice

## Example

```
Cline, create a project directory with:
- index.html (landing page)
- styles/main.css
- scripts/app.js
Then zip it for Dropley upload.
```
