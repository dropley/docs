# Publishing with Gemini

Google Gemini can help you create projects and prepare them for Dropley publishing.

## Prerequisites

- Access to Google Gemini (gemini.google.com or API)

## Workflow

1. Ask Gemini to generate your project
2. Request the output as files you can download
3. Package into a ZIP (Gemini may provide individual files)
4. Upload the ZIP to [dropley.app](https://dropley.app)

## Pro Tips

- Ask Gemini for a complete project structure with multiple files
- Specify that paths should be relative for static hosting
- Gemini can explain the ZIP structure if you need to create it manually

## Example

```
Create a landing page for a product launch. Give me the HTML, CSS,
and JavaScript as separate files with relative paths so I can
package them as a ZIP for Dropley.
```
