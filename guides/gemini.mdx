# Publishing with Gemini

Gemini can publish artifacts to Dropley directly.

## Setup

Tell Gemini to install the Dropley skill:

```
npx skill add dropley/docs
```

## Workflow

1. Ask Gemini to generate your project
2. Tell Gemini: "Publish this to Dropley"
3. Gemini uses the skill to upload your files and return a shareable URL

## Example

```
Create a landing page for a product launch with index.html, styles/main.css,
and scripts/app.js. Use relative paths. Then publish to Dropley.
```
