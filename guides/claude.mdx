# Publishing with Claude

Claude can help you create and publish artifacts to Dropley directly from your conversation.

## Prerequisites

- Claude (claude.ai or API)
- Your project files ready

## Workflow

1. Ask Claude to generate your project (HTML page, dashboard, etc.)
2. Tell Claude: "Package this as a ZIP file I can upload to Dropley"
3. Claude will prepare the ZIP with the correct structure
4. Download the ZIP from Claude
5. Upload to [dropley.app](https://dropley.app)

## Pro Tips

- Ask Claude to structure your project as a proper ZIP before downloading
- For single-page projects, ask for inline CSS/JS so everything is in one file
- Claude can also generate the `index.html` ent ry point for multi-file projects

## Example Prompt

```
Create a personal portfolio page as a ZIP file for Dropley.
Include: index.html, styles/main.css, and a projects listing.
Make sure all paths are relative.
```

Claude will generate the files and package them into a downloadable ZIP.
